# CPT Admin Column Utility

A simple, procedural PHP utility for adding custom, sortable columns to the admin list table for WordPress Custom Post Types.

This utility is designed to be included in a WordPress theme (e.g., in `functions.php`) and provides a clean and configurable way to enhance the CPT admin experience.
